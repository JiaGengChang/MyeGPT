Table name: `chromothripsis`
Description: Calls for chromothripsis, whole genome shattering, as detected by Shatterseek.
Below is a description of each field.
column_number | name | description
1	PUBLIC_ID	Patient public identifier. Last 4 digits are unique. e.g. MMRF_1234
2   chromothripsis   Indicates whether chromothripsis occurred as per the ShatterSeek algorithm at significance level of 0.05. There are three kinds of values: `negative` means not occurred, `lowconf` means low confidence, and `highconf` means high confidence. Only `highconf` cases should be taken as positive. `negative` and `lowconf` cases are taken to be negative in chromothripsis.
