Table name: `per_visit`
Description: Clinical data obtained from each visit made by a patient. Each patient has multiple visits throughout the treatment journey and results for each visit are recorded here. If column description is Ignore, do not use the column.
Format: column_number | column_name | column_description
# Identifier columns
1|public_id|Index column. Patient public identifier. MMRF_ followed by 4 unique digits. e.g. MMRF_1234
3|visit|A unique number assigned to the visit. e.g. 1 means first visit at which diagnosis was made.
5|visitdy|Day of Visit counting from day of diagnosis
# Sample and specimen metadata fields
356|sample_type|Either,serial (same as previous), BM (bone marrow) baseline, or PB (peripheral blood).
357|specimen_id|An ID ,or the specimen, of the format [PUBLIC_ID]_[VISIT_ID]_[TISSUE SITE: BM or PB]. e.g., MMRF_2464_1_BM
